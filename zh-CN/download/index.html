通过 Docker 安装​

GeaFlow Docker 镜像可在 Docker Hub 上获取

geaflow​

geaflow-0.6-java8-x86

docker pull geaflow/geaflow:0.6（获取 GeaFlow 镜像：geaflow/geaflow:0.6）

geaflow-0.6-java8-arm

获取 GeaFlow 镜像：geaflow/geaflow-arm:0.6

GeaFlow 控制台​

geaflow-console-0.6-java8-x86

获取 GeaFlow 控制台镜像：geaflow/geaflow-console:0.6


geaflow-console-0.6-java8-arm

获取 GeaFlow 控制台镜像：geaflow/geaflow-console-arm:0.6

Apache GeaFlow GeaFlow K8S 控制器​

geaflow-kubernetes-operator​

geaflow-kubernetes-operator-0.6-java8-x86

获取 GeaFlow K8S 控制器镜像：geaflow/geaflow-kubernetes-operator:0.6
