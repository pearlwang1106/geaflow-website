Installing with Docker​

GeaFlow docker images are available from Dockerhub Official Images.

geaflow​

geaflow-0.6-java8-x86

docker pull geaflow/geaflow:0.6

geaflow-0.6-java8-arm

docker pull geaflow/geaflow-arm:0.6

geaflow-console​

geaflow-console-0.6-java8-x86

docker pull geaflow/geaflow-console:0.6

geaflow-console-0.6-java8-arm

docker pull geaflow/geaflow-console-arm:0.6

Apache GeaFlow Kubernetes Operator​

geaflow-kubernetes-operator​

geaflow-kubernetes-operator-0.6-java8-x86

docker pull geaflow/geaflow-kubernetes-operator:0.6

Maven Dependencies​
